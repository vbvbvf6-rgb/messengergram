# Deploy 24/7 (free, worldwide)

Этот проект можно развернуть на бесплатном хостинге без карты. Для вашего случая лучший вариант — `Replit`, потому что он работает без карты и подходит для Python/Socket.IO.

## Почему Replit
- Регистрация без карты
- Поддерживает Python-проекты и веб-сервера
- Может запускать `python main.py`
- Даёт публичный URL
- Отлично подходит, если Koyeb требует карту

## Почему Koyeb
- Koyeb раньше работал без карты, но сейчас может запросить платёжные данные
- Если Koyeb требует карту, используйте Replit или другую бесплатную платформу
- Для гарантированной 24/7 работы без платы потребуется другой подход
- Важно: на бесплатном тарифе всегда есть ограничения

## Шаги установки

### 1. Создать аккаунт
1. Перейди на https://www.koyeb.com
2. Зарегистрируйся по email или через GitHub
3. Подтверди аккаунт по email

### 2. Создать приложение
1. Открой Dashboard Koyeb
2. Нажми "Create App"
3. Выбери "Git Repository"
4. Подключи репозиторий GitHub
5. Укажи ветку `main`

### 3. Настроить приложение
- `App Name`: `droidgram`
- `Region`: ближайший к тебе, например `FRA` или `AMS`
- `Runtime`: Python 3.11
- `Build Command`: оставить пустым
- `Start Command`: `python main.py`
- `HTTP Port`: `8000`

### 4. Добавить переменные окружения
Если используешь PostgreSQL, добавь секрет:
- `DATABASE_URL` = `postgresql://user:pass@host/db`

Если не используешь БД, оставь переменные пустыми.

### 5. Деплой приложения
1. Нажми "Create App" и подожди завершения сборки
2. Koyeb автоматически запустит приложение
3. URL будет выглядеть как `https://droidgram.<region>.koyeb.app`

### 6. Проверка
Открой в браузере:
```text
https://<app-name>.<region>.koyeb.app/static/index.html
```

## Если Koyeb не грузится
Иногда сайт Koyeb может быть временно недоступен или он может потребовать платёжную карту. В таком случае лучше перейти на другой бесплатный сервис без карты.

### Бесплатная альтернативы без карты
- `Replit.com` — позволяет запускать Python-сервер и даёт публичный URL
- `Deta.Space` — бесплатный хостинг без карты (но с ограничениями для длительных соединений)
- `Glitch.com` — можно использовать для простых веб-проектов

> Ни один бесплатный сервис без карты не даёт честной гарантии «максимум 24/7». Бесплатный хост может засыпать или иметь ограничения.

## Как запустить на Replit
1. Зайди на https://replit.com
2. Зарегистрируйся по email или через GitHub
3. Создай новый Repl и подключи репозиторий GitHub
4. Выбери `Python`
5. Укажи команду запуска:
```bash
python main.py
```
6. В настройках Replit добавь переменную окружения `PORT` со значением `8000`
7. Запусти проект и возьми публичный URL

## Запасной вариант: локальный сервер без карты
Если с удалённым хостингом сложно, разверни приложение на своём компьютере и используй туннель только как временный доступ:

1. Установи Python и запусти приложение локально:
```bash
python main.py
```
2. Открой доступ через публичный туннель (работает бесплатно и без карты):

### Вариант A: localtunnel
```bash
npm install -g localtunnel
lt --port 8000
```

### Вариант B: Cloudflare Tunnel
```bash
# скачай cloudflared для Windows с официального сайта и запусти
cloudflared tunnel --url http://localhost:8000
```

3. Возьми сгенерированный URL и отправь друзьям.

> Такой способ работает бесплатно и без карты, но компьютер должен быть включён.

## База данных
- **Рекомендуется**: Back4App PostgreSQL или другой бесплатный PostgreSQL хост
- **Можно**: SQLite, но данные будут храниться только на текущем сервере

Если используешь PostgreSQL, добавь `DATABASE_URL` в Secrets Koyeb.
